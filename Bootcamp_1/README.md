## Содержание проекта

Данный проект состоит из различных задач, направленных на изучение базовых конструкций Python,
работы с файлами, алгоритмами и простыми структурами данных.

### Задание 1

Вычислить скалярное произведение двух векторов в 3х мерном пространстве.

Для ввода/вывода использовать стандартный поток. Корректность входных данных проверять не надо

- Входные данные: Вещественные числа, координаты 2х векторов на 2х строках
- Выходные данные: Вещественное число, скалярное произведение векторов. 
  
| Входные данные        | Выходные данные |
|-----------------------|-----------------|
| `1.0 2.0 3.0`<br>`4.0 5.0 6.0` | `32.0` |


### Задание 2

Определить является ли число палиндромом или нет.

Для ввода/вывода использовать стандартный поток. Строки использовать нельзя.
Отрицательные числа не считать палиндромами.

- Входные данные: Целое число
- Выходные данные: True/False (палиндром/не палиндром)

| Входные данные | Выходные данные |
|----------------|-----------------|
| `1143411`      | `True`          |

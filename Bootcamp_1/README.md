### Задание 7

Робот умеет перемещаться либо вниз, либо вправо на 1 клетку поля. Поле прямоугольное и содержит числа - это кол-во монет в каждой клетке поля. Робот собирает монеты из каждой клетки по которой прошел. Изначальное расположение - левая верхняя клетка поля, он гарантированно соберет монеты в ней. 
Задача робота - собрать максимальное кол-во монет пока он направляется к правой нижней клетке поля, поэтому движение всегда происходит по наболее выгодному маршруту.

Определить по полю склолько монет будет собрано роботом.

Ввод-вывод происходит через стандартный поток, корректность входных данных проверять не стоит. 

- Входные данные: 2 натуральных числа - кол-во строк и столбцов. Также строки которые содержат неотрицательные числа - кол-во монет в каждой клетке поля.
- Выходные данные: Неотрицательное число - суммарное кол-во монет которое будет собрано роботом.

| Входные данные | Выходные данные  |
|:----------|:----------|
| 3 4</br>3 0 2 1</br>6 4 8 5</br>3 3 6 0     |27    |

### Задание 9

Посчитать производную многочлена в точке. Вывести результат с 3мя знаками после запятой. Для ввода/вывода использовать стандартный поток. Корректность входных данных проверять не надо.

- Входные данные: В первой строке содержится натуральное число - это старшая степень многочлена. Также содержится вещественное число - это точка в которой надо найти производную. Далее строки содержат вещественные числа - это коэффициенты при степенях х, начиная со старшей степени.

- Выходные данные: Вещественное число, производная многочлена в точке.

К примеру:
Многочлен - `5 * x**2 + 1.2 * x - 3`
Производная - `10 * x + 1.2`
Производная в точке - `3.0` : `30 + 1.2 = 31.2`

| Входные данные  | Выходные данные  |
|:----------|:----------|
| 2 3.0</br>5</br>1.2</br>-3</br>    | 31.200    |
